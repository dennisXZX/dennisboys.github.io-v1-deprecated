Binary Search

Posted on 2016-04-17   |   In Java   |  

For binary search to work, the array must be sorted first.

Pseudo-code for binary search

get the min value
get the max value

while max value greater than and equal to min value

calculate the mid value

if the search value is greater than the mid value
    set min to mid + 1
if the search value is less than the mid value
    set max to mid - 1
else
    return mid value

return -1
